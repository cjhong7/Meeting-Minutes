📂 협의록 불러오기

        
      

      

        

          
        

        

          
저장된 회의록이 없습니다.

            협의록을 생성하면 자동으로 보관함에 저장됩니다.
          

        

      

    

  


  
  

    

      

        
☁️ 구글 드라이브·원드라이브 연동

        
      

      

        
PC의 클라우드 동기화 폴더를 보관함으로 지정하면,
          협의록이 자동으로 클라우드에 백업됩니다. (별도 로그인 불필요)


        
🟢 구글 드라이브

        
    
          
  1. Google Drive for Desktop 설치
            (google.com/drive/download)
    2. 
          
  2. 설치 후 내 PC에 드라이브 폴더가 생깁니다
            (예: G:\내 드라이브)
    3. 
          
  3. 이 앱에서 [📁 보관함 폴더 지정] → 그 드라이브 폴더 선택
    4. 
          
  4. 이제 협의록 생성 시 자동으로 그 폴더에 저장 → 구글 드라이브가 자동 업로드 ☁️
    5. 
        


        
🔵 원드라이브 (OneDrive)

        
    
          
  1. Windows에 기본 설치된 OneDrive 로그인
    2. 
          
  2. 내 PC의 OneDrive 폴더 확인 (예: C:\Users\사용자\OneDrive)
    3. 
          
  3. 이 앱에서 [📁 보관함 폴더 지정] → OneDrive 폴더 선택
    4. 
          
  4. 협의록 생성 시 자동 저장 → OneDrive가 자동 백업 ☁️
    5. 
        


        

          
💡 장점

          
    
            
  • 저장 즉시 클라우드 자동 백업 (PC 고장에도 안전)
    • 
            
  • 휴대폰·다른 PC에서 드라이브 앱으로 협의록 열람 가능
    • 
            
  • 협의록이 개발자 서버를 거치지 않아 안전
    • 
          

          
⚠ 폴더 지정은 데스크톱 크롬·엣지에서만 가능합니다.
            (휴대폰·태블릿은 다운로드 폴더에 저장됩니다)
